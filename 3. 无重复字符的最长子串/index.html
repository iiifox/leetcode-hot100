「力扣」第 3 题：无重复字符的最长子串

  • 链接

题目描述

给定一个字符串 s ，请你找出其中不含有重复字符的 最长子串 的长度。

 

示例 1:
输入: s = "abcabcbb"
输出: 3 
解释: 因为无重复字符的最长子串是 "abc"，所以其长度为 3。

示例 2:
输入: s = "bbbbb"
输出: 1
解释: 因为无重复字符的最长子串是 "b"，所以其长度为 1。

示例 3:
输入: s = "pwwkew"
输出: 3
解释: 因为无重复字符的最长子串是 "wke"，所以其长度为 3。
     请注意，你的答案必须是 子串 的长度，"pwke" 是一个子序列，不是子串。

 

提示：

  • 0 <= s.length <= 5 * 104
  • s 由英文字母、数字、符号和空格组成

 

 

参考代码

public class Solution {
    public int lengthOfLongestSubstring(String s) {
        int len = s.length();
        if (len < 2) {
            return len;
        }

        // 由于 s.charAt(i) 操作会去检查下标是否越界，因此
        // 在 Java 里，将字符串转换成字符数组是常见的操作
        char[] chars = s.toCharArray();

        // 窗口数组，用来代替 HashMap，提高运行效率
        // windows[i] 存储的是 chars 中字符值为 i 的最新索引
        int[] window = new int[128];
        for (int i = 0; i < 128; i++) {
            window[i] = -1;
        }

        int res = 0;
        // chars 在 [left, right] 中没有重复元素
        for (int left = 0, right = 0; right < len; right++) {
            // 字符值为 chars[right] 在前面出现过了，
            // 前一次出现的索引是否在 [left, right] 中决定是否进行窗口滑动
            if (window[chars[right]] != -1) {
                left = Math.max(left, window[chars[right]] + 1);
            }
            // 更新一下字符值为 chars[right] 的最新索引
            window[chars[right]] = right;
            // 计算 [left, right] 区间长度，决定是否更新 res
            res = Math.max(res, right - left + 1);
        }
        return res;
    }
}
