「力扣」第 4 题：寻找两个正序数组的中位数

  • 链接

题目描述

给定两个大小分别为 m 和 n 的正序（从小到大）数组 nums1 和 nums2。请你找出并返回这两个正序数组的 中位数 。

算法的时间复杂度应该为 O(log (m+n)) 。

 

示例 1：
输入：nums1 = [1,3], nums2 = [2]
输出：2.00000
解释：合并数组 = [1,2,3] ，中位数 2

示例 2：
输入：nums1 = [1,2], nums2 = [3,4]
输出：2.50000
解释：合并数组 = [1,2,3,4] ，中位数 (2 + 3) / 2 = 2.5

 

提示：

  • nums1.length == m
  • nums2.length == n
  • 0 <= m <= 1000
  • 0 <= n <= 1000
  • 1 <= m + n <= 2000
  • -106 <= nums1[i], nums2[i] <= 106

 

 

分析：

在两个有序数组中间画一条红线，这条红线满足如下条件：

  • 条件①：红线左边所有的数都小于等于红线右边所有的数；
  • 条件②：size(左) = size(右) 或 size(左) = size(右) + 1

记 m = nums1.length, n = nums2.length

  • m + n 为奇数：size(左) = size(右) + 1，中位数即红线左边最大值
  • m + n 为偶数：中位数即红线左边最大值与右边最小值的均值

我们记：

  • 数组 nums1 在红线右边的第 1 个数的索引为 i
  • 数组 nums2 在红线右边的第 1 个数的索引为 j

此时恒有 size(左) = i + j = (m + n + 1) >>> 1

此时问题转化为在区间 [0, m] 中寻找索引 i，使得条件①和②成立

 

参考代码

public class Solution {
    public double findMedianSortedArrays(int[] nums1, int[] nums2) {
        // 为了让搜索范围更小，我们始终让 num1 是那个更短的数组
        if (nums1.length > nums2.length) {
            int[] temp = nums1;
            nums1 = nums2;
            nums2 = temp;
        }

        // 上述交换保证了 m <= n，在更短的区间 [0, m] 中搜索，会更快一些
        int m = nums1.length;
        int n = nums2.length;

        // 使用二分查找在区间 [left, right] 中寻找索引 i，使得条件①和②成立
        int left = 0;
        int right = m;
        // size(左)，可通过该数求得 j = totalLeft - i;
        int totalLeft = (m + n + 1) >>> 1;
        while (left < right) {
            // 下面的 if 中左指针右移，故而需要向下取整
            int i = (left + right) >>> 1;
            int j = totalLeft - i;
            // nums2 左边的最大值比 nums1 右边的最小值要大，说明 i 需要扩大
            if (nums2[j - 1] > nums1[i]) {
                left = i + 1;
            } else {
                // 左边的最大值小于等于 nums1 右边的最小值，因此 i 不需要扩大
                right = i;
            }
        }

        // 退出循环时的 i = (left = right) 即为满足条件①和②的 i
        int i = left;
        int j = totalLeft - i;

        // 边界情况需要特殊考虑
        int nums1LeftMax = (i == 0) ? Integer.MIN_VALUE : nums1[i - 1];
        int nums1RightMin = (i == m) ? Integer.MAX_VALUE : nums1[i];
        int nums2LeftMax = (j == 0) ? Integer.MIN_VALUE : nums2[j - 1];
        int nums2RightMin = (j == n) ? Integer.MAX_VALUE : nums2[j];

        return (((m + n) & 1) == 1) ?
                Math.max(nums1LeftMax, nums2LeftMax) :
                (double) (Math.max(nums1LeftMax, nums2LeftMax) +
        Math.min(nums1RightMin, nums2RightMin)) / 2;
    }
}
